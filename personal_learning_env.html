Operating Systems

Most developers use Unix-like systems due to tooling, package managers, and server parity.

  • Linux: Maximum control and compatibility
  • macOS: Unix-based with strong commercial tooling
  • Windows: Best with WSL for development

Development Tools & IDEs

IDEs and editors improve productivity through refactoring, debugging, and code intelligence.

  • VS Code (most common)
  • JetBrains IDEs
  • Vim / Neovim / Emacs

Version Control (Git & GitHub)

Version control is mandatory for collaboration and personal learning.

  • Git fundamentals: commits, branches, merges
  • GitHub for hosting and collaboration
  • Pull requests and code reviews

Productivity & Note-Taking

Learning efficiency improves when knowledge is captured, reviewed, and connected.

  • Markdown notes (Obsidian, Notion)
  • Task tracking (Todo lists, Kanban)
  • Spaced repetition for retention
